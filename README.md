## Features
- Increase, decrease, and reset the counter.
- Fully responsive UI.
- Built with HTML, CSS, and JavaScript.

## How to Use
1. Click the **"+"** button to increase the count.
2. Click the **"-"** button to decrease the count.
3. Click **Reset** to set the counter back to zero.
